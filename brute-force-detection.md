## 🔐 Brute Force Attack Detection

### 📌 Project Overview
Simulated detection of brute force login attempts by analyzing repeated failed login activity.

### 🔍 What I Did
- Analyzed login logs for multiple failed attempts from the same user/IP
- Identified patterns indicating brute-force attacks
- Flagged suspicious accounts after repeated failures
- Recommended mitigation actions such as account lockout and MFA

### 🛠️ Tools & Skills
Log Analysis, Threat Detection, Python (basic), Security Monitoring

### 🎯 SOC Relevance
Brute force attacks are common in real environments. This project demonstrates how SOC analysts detect repeated login failures and respond to potential account compromise.
