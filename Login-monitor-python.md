## 🖥️ Login Monitoring Automation (Python)

### 📌 Project Overview
Built a Python script to simulate SOC analyst login monitoring and access control.

### 🔍 What I Did
- Checked if users are approved using an allow list
- Verified login attempts during organization hours
- Used conditional logic (`if`, `elif`, `and`, `in`) to evaluate login activity
- Flagged unauthorized users and off-hours login attempts

### 🛠️ Tools & Skills
Python, Access Control, Security Monitoring, Conditional Logic

### 🎯 SOC Relevance
This project demonstrates how SOC analysts monitor authentication activity, validate authorized users, and detect suspicious login behavior such as unauthorized access or unusual login times.
